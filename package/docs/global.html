(constant) FileUpload

Description:

  • ✅ Single File Upload Component

    Displays a file input with optional label, preview (for images), error message, and helper text. Supports drag-and-drop. Behavior changes based on type prop from parent:

    • 'photo': Accepts images only, always shows preview.
    • 'document': Accepts documents only, shows file name with icon, no preview.
    • 'general': Accepts any file, preview if image, else file name. Parent can pass separate onChange handlers for photo vs. document logic if needed.
